郭采洁

郭采洁（Amber Kuo），1986年2月19日出生于台湾省台北市，中国台湾影视女演员、流行乐歌手。 2007年，发行首张音乐专辑《隐形超人》而正式出道 [1] 。2008年，出演个人首部电视剧《无敌珊宝妹》获得关注，并获得台湾偶像剧年度最佳新人奖 [2-3] 。2009年主演偶像剧《那一年的幸福时光》，凭该片提名第45届台湾电视金钟奖戏剧节目女主角奖 [4] ；同年，发行第二张音乐专辑《爱异想》。2010年出演首部电影作品《一页台北》，并凭该片获得第12届台北电影节最佳新演员奖 [5] ；同年，主演爱情片《近在咫尺的爱恋》 [6-7] 。 2012年，郭采洁因在电影《小时代》中饰演顾里受到广泛关注，并凭该片提名第16届上海国际电影节电影频道传媒大奖最佳女主角 [8] ；同年，主演都市爱情片《意外的恋爱时光》 [9] 。2014年首次登上央视春晚，并演唱歌曲《答案》；同年，主演了爱情片《冲上云霄》 [10] 。2015年，发行第五张音乐专辑《爱造飞鸡》 [11] ；同年，主演真人CG魔幻题材电影《爵迹》；2016年，主演动作片《冲天火》 [12] 。

早年经历

2005年，郭采洁参加国立台北大学学校歌唱比赛并获得第六届北韵奖。大学一年级时，郭采洁翻唱艾薇儿的歌曲《Why》，夺得个人组冠军而被发掘成为歌手。此外，郭采洁在出道前曾参与Tank《我们小时候》MV的演出，而在成为歌手前，郭采洁是一名平面、电视广告模特 [13-15] 。

演艺经历

006年，郭采洁签约了娱乐产房经纪公司，并接受了一年半的培训 [15] 。

2007年12月25日，推出个人首张音乐专辑《隐形超人》，以“优格女孩”为号出道。专辑收录了第一主打《隐形超人》以及插曲《I Remember》等在内的十首歌曲 [1] [16] ；同年，郭采洁被制作人选中，并出演了爱情剧《无敌珊宝妹》中女主角胡珊宝一角 [3] 。

2008年6月7日，郭采洁于国立台北大学社工系毕业 [17] ；8月24日，郭采洁主演的爱情剧《无敌珊宝妹》在台视首播，该剧播出后首集收视分段最高达10.57，总平均7.25，打破了偶像剧开播收视纪录，而郭采洁个人则凭借该角色获得关注，并获得台湾偶像剧年度最佳新人奖 [2-3] 。

2009年5月1日，发行第二张个人音乐专辑《爱异想》。该专辑以千变万化的爱情为主题，收录了爱的悲歌《又圆了的月亮》、失恋的《你在，不在》以及抒情的《狠狠哭》等在内的十首歌曲 [18-19] 。而郭采洁个人则凭借该专辑入围东南劲爆音乐榜“劲爆港台地区最佳新人奖” [20] ；8月7日，郭采洁主演的台湾偶像剧《那一年的幸福时光》播出，她在剧中饰演甜美可爱、清丽脱俗的乡下老师陈素馨，并凭该角色提名第45届台湾电视金钟奖戏剧节目女主角奖 [4] [21-22] 。

2010年4月2日，郭采洁主演的个人首部电影作品《一页台北》上映，她在片中饰演清新可人的书店店员Susie，凭借该片中的表现她获得了第12届台北电影节最佳新演员奖 [5] [15] ；6月9日，发行第三张音乐专辑《烟火》，而专辑则收录了同名主打歌《烟火》以及插曲《派乐地》、《爱不爱》等在内的11首歌曲，该专辑在第九届MusicRadio中国TOP排行榜上获得“港台地区推荐唱片”，歌曲《烟火》亦获得港台地区年度最佳金曲 [23-24] ；8月13日，其与明道、彭于晏合作出演的爱情片《近在咫尺的爱恋》在中国台湾上映，郭采洁在片中饰演率性直爽、单纯的丁小葵 [6-7] ；12月5日，郭采洁与郑元畅合作出演的惊悚推理偶像剧《国民英雄》播出，她在剧中饰演狗仔记者洪小绿 [25] 。

2012年，郭采洁开始参演根据郭敬明长篇小说改编的《小时代》系列电影 [8] ；2月13日，郭采洁主演的由钮承泽执导的爱情片《LOVE》上映，她在片中饰演了聪明有主见的小霓，而郭采洁个人则凭借该角色提名第49届台湾电影金马奖最佳女配角奖，并在第14届台北电影节夺得最佳女配角奖 [26] ；2月19日，青春偶像剧《向前走向爱走》播出，郭采洁在剧中饰演了倔强孤傲、刁蛮霸道的任晶晶 [27] ；8月10日，推出第四张音乐专辑《给他》，而专辑则收录了同名主打《给他》以及插曲《灌溉爱》、《该忘了》等在内的十首歌曲 [28] 。

2013年6月27日，郭采洁搭档杨幂、郭碧婷等主演的都市青春片《小时代》上映，她在片中饰演冷酷聪明、霸道强势，永远是高高在上的女王姿态的冷艳女王顾里。该片上映首日票房约7300万元人民币，同时也打破了中国2D电影的首日票房纪录 [29-30] ；8月8日，其主演的都市青春片《小时代2：青木时代》上映，该片则取得2.9亿元的票房成绩 [31] ；11月8日，郭采洁在上映的都市爱情片《意外的恋爱时光》中饰演了傲娇、美丽的王乐晴 [32] 。

2014年1月30日，郭采洁首次登上中央电视台春节联欢晚会，并与杨坤合唱歌曲《答案》 [10] ；7月17日，郭采洁主演的《小时代3：刺金时代》上映，该片取得4.83亿元的票房成绩 [33] ；10月24日，剧情片《不能说的夏天》上映，郭采洁在片中饰演无助、楚楚可怜的音乐系女研究生白白 [34] 。

2015年2月9日，郭采洁与古天乐、张智霖等合作主演的爱情片《冲上云霄》上映，她在片中饰演性格率真的薛健雅；6月26日，郭采洁在上映的爱情片《对风说爱你》中饰演女主角邱梅 [35] ；7月9日，郭采洁主演的电影《小时代4：灵魂尽头》上映首日破亿 [36] ；7月31日，其主演的爱情喜剧片《巴黎假期》上映，郭采洁在片中饰演自暴自弃的留学画家丁晓敏 [37] ；11月26日，郭采洁主演的由张家辉执导的恐怖片《陀地驱魔人》上映 [38] ；12月25日，推出第五张音乐专辑《爱造飞鸡》迷你电影原声带，专辑收录了电影同名主题曲《Bird n Tree》以及插曲《单飞》、《外面》以及等在内的12首歌曲 [11] 。

2016年2月5日，郭采洁在上映的动作喜剧片《我不做大哥好多年》中饰演了热情奔放的朱小芹 [39] ；9月30日，全真人CG魔幻题材电影《爵迹》上映，郭采洁在片中饰演性感但却十分残酷的现任四度王爵特蕾娅 [40] ；11月25日，由林岭东执导的悬疑动作片《冲天火》上映，郭采洁在片中饰演具有艺术天分的小珍 [12] 。

2017年2月4日，与陆毅、炎亚纶合作主演了都市喜剧《亲·爱的味道》，饰演了女主靳津津；2月17日，其主演的动作片《上海王》上映，该片由胡雪桦执导，郭采洁在片中饰演洪门第二代、常力雄与筱月桂之女常荔荔 [41] ；7月17日，其主演的喜剧片《绝世高手》上映，郭采洁在片中饰演天性直率的小曼 [42] 。

2018年4月1日，与陈思成、佟丽娅、袁弘合作主演的年代传奇剧《远大前程》首播，剧中饰演出身名门的于梦竹 [43] ；9月，主演根据亦舒同名小说改编的电影《喜宝》 [44] 。

2019年5月1日，郭采洁搭档郑恺主演的爱情片《下一任：前任》上映，片中饰演林心恬 [45] 。

个人生活

家庭

郭采洁在家中排行老二，在就读高中前本名郭彦君，后毕业于台北市立松山高级中学、国立台北大学社会工作学系。她坦言因为女孩子的名字里带君就要担起养家的重担，家里不想让她有负担，加上高中时期流行换名字，所以家里每个人都换了名字 [15] [46] 。

感情

2011年，郭采洁在拍摄电视剧《向前走向爱走》时与男演员杨祐宁开始交往 [47] 。2015年3月16日，两人分手 [48] 。

社会活动

2008年8月29日，郭采洁担任青苹果大使，为参与青苹果项目的同学们加油打气，鼓励学生努力向上 [46] 。

2009年10月，郭采洁担任交通部机车安全宣导广告短片的公益大使；12月9日，出席关怀爱滋拒绝毒品创意倡导成果记者会，并担任2010防爱滋反毒代言人 [59] 。

2010年5月16日，郭采洁受邀担任99年反毒大使，呼吁“全民总动员、无毒保家园” [60] ；10月，担任喜憨儿基金会公益大使 [61] ；同年，担任内政部全国孝行奖孝行大使。

2012年1月7日，郭采洁与彭于晏前往非洲献爱心，并担任台湾世界展望会第23届“饥饿30”代言人 [62] 。

2013年5月27日，郭采洁担任卫教大使，为失智动画《阿嬷妮》失智症十大警讯宣导动画配音 [63] ；8月，出席了为“孩想要飞 用爱温饱”的弱势儿童开学扶助计划公益并担任爱心大使，鼓励更多人去关心和扶持弱势儿童 [64] ；9月，郭采洁出席第二届“幸福袋着走，让爱无阻碍”公益活动的首卖会并担任幸福大使，呼吁更多人去关注和帮助智障青年的生存情况 [64-65] 。

2017年9月9日，郭采洁参加了第15届芭莎明星慈善夜，并与众多艺人共同为“为爱加速”思源﹒芭莎自治区救护车项目募集善款 [66] 。

2018年4月25日，担任“中国宫颈癌预防控制科普项目”宣教大使的郭采洁为该公益项目演唱了公益歌曲《这样的我》 [67] 。
